# Sistema de ProdutosWeb
> Aplicação feita para cadastro e consulta de produtos.

## Projeto desenvolvido em Java Spring MVC
1. Spring MVC
2. Maven
3. Spring JDBC
4. Bootstrap
5. JQuery
